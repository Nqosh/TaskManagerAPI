
A full-stack task management system featuring JWT auth, task CRUD, filtering, and real-time updates with SignalR.

## Stack
- **Backend**: .NET 8, ASP.NET Core Web API, EF Core (SQLite for dev), JWT, SignalR, AutoMapper, xUnit
